# OMNIX - the ultimate NixOS configuration

`badges here`

**OMNIX makes it easy to maintain, distribute and manage** your NixOS configs -
no matter if you're an advanced NixOS user or just starting out with your very
first config. It provides a structured way to manage all of your systems
configuration and dotfiles with minimal effort. It includes a _home-manager_,
_secrets_, _community configurations_ with _custom themes_, diffrent
_desktop environments_ and much much more!

**Features:**

- Quick start scripts: These scripts allow you to quickly get into actually using
  your system without the need for extensive configuration.
- Easy community configs support: You can just drag and drop any OMNIX compatible
  configuration override into the `./overrides` folder, rebuild the system and be
  using it in no time!
